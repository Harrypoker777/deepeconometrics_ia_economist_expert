# Lawrence R. Klein (Nobel de Economía 1980)

Pionero en la construcción de modelos macroeconométricos a gran escala.
Recibió el premio por crear modelos empíricos usados para analizar fluctuaciones
económicas y políticas económicas.

## Ideas clave

- **Modelos Klein-Goldberger.** Sistemas de ecuaciones simultáneas estimados
  con datos históricos, base conceptual para los modelos DSGE y VAR posteriores.
- **Project LINK.** Vinculó modelos nacionales en un sistema mundial para
  estudiar transmisión internacional de shocks.
- **Proyecciones a corto plazo.** Integración de insumos econométricos con
  juicio experto para afinar pronósticos trimestrales.
- **Identificación en ecuaciones simultáneas.** Desarrollo de técnicas de
  estimación (2SLS, 3SLS) para sistemas interdependientes.

## Uso en respuestas

Es la referencia natural cuando el usuario pregunte sobre **construcción de
modelos econométricos, forecasting macro, sistemas de ecuaciones simultáneas o
análisis de impacto de políticas**. DeepEconometrics puede validar la
racionalidad de un forecast contrastándolo con la lógica de un modelo
estructural al estilo Klein, aunque internamente use una especificación
reducida (OLS, VAR, etc.).
